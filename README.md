# 📊 Amazon Sales Dashboard 2025

## 🔎 Project Overview
This project presents an interactive Sales Dashboard built entirely in Power BI to analyze Amazon sales data.  
The dashboard provides insights into sales performance, product trends, payment methods, delivery status, and geographic distribution.

---

## 🎯 Business Objective
The main objective of this project is to:
- Monitor overall sales performance
- Identify top-performing products and categories
- Analyze customer payment preferences
- Track order and delivery status
- Generate business insights for decision-making

---

## 🛠 Tools & Technologies Used
- Power BI
- Power Query (Data Cleaning & Transformation)
- DAX (Data Analysis Expressions)
- Interactive Visualizations

---

## 📈 Key KPIs
- **Total Sales:** 240K
- **Total Quantity Sold:** 693
- **Total Products:** 244

---

## 📊 Dashboard Insights

### 🔹 Sales by Category
- Electronics is the highest revenue-generating category (127K)
- Home Appliances follow with strong performance (105K)

### 🔹 Top 5 Products
1. Refrigerator (78K)
2. Laptop (56K)
3. Smartphone (49K)
4. Washing Machine (27K)
5. Smartwatch (15K)

### 🔹 Payment Analysis
- PayPal has the highest transaction share
- Credit & Debit Cards show balanced usage
- Amazon Pay usage is moderate

### 🔹 Delivery Status
- Majority of orders are Completed or Pending
- Cancellation rate is comparatively lower

### 🔹 Geographic Distribution
- Houston and Miami generate higher sales compared to other cities
